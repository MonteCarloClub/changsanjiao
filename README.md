## 运行

### 配环境

本地有 node 和 npm、pnpm 的环境，运行以下指令查看版本

``` bash
$ node -v
$ npm -v
$ pnpm -v
```

如果 `pnpm -v` 报错的话，就手动全局安装一下 pnpm（理解为 npm 的升级版）

```bash
$ npm install -g pnpm
$ pnpm -v
```

### 跑项目

``` bash
$ pnpm install
$ pnpm run dev
```
