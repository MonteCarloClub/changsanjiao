# 长三角学分银行功能演示

## 学习成果上链

剧本：

1. 表单里输入个人信息
2. 缩小为抽象形状，代表数据
3. 数据进入某个节点
4. 节点广播数据至其它节点


## 运行

### 配环境

本地有 node 和 npm、pnpm 的环境，运行以下指令查看版本

``` bash
$ node -v
$ npm -v
$ pnpm -v
```

如果 `pnpm -v` 报错的话，就手动全局安装一下 pnpm（理解为 npm 的升级版）

```bash
$ npm install -g pnpm
$ pnpm -v
```

### 跑项目

``` bash
$ pnpm install
$ pnpm run dev
```
